# Pipeline ETL com Python e IA Generativa

## Descrição
Projeto desenvolvido para o desafio "Explorando IA Generativa em um Pipeline de ETL com Python".

O objetivo é demonstrar o funcionamento de um pipeline ETL (Extract, Transform e Load) utilizando Python e simulação de IA generativa para criação de mensagens personalizadas.

## Etapas do Pipeline

### Extract
Criação de uma lista de usuários fictícios diretamente no código.

### Transform
Geração de mensagens personalizadas simulando o uso de IA generativa.

### Load
Exibição das mensagens geradas no console.

## Tecnologias Utilizadas
- Python
